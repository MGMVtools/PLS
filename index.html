About

This is an online tool developed so that anyone can use the Partial Least Squares algorithm (Herman Wold, "Partial Least Squares") without needing to install any software.

This tool can be used for educational purposes, testing, or developing computer software at free will, regarding that due credit is provided.

This website does not collect any third- or second- party data, neither do I (the author) collect or get access to any data. The code is run using your browser built-in javascript interpreter and no data is upload to any server. Use this tool at your responsibility, I will not be held responsible for any problem encountered with the code. The most important step in any data analysis technique is to interpret the results and check their validity.

Nevertheless, if you find anything that needs to be fixed, or any suggestion that you would like to make, do not hesitate contact me so that I can further improve this tool.
